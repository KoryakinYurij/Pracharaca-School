# Ограничения и запреты

## Что НЕ делаем

Это **не платформа**:

- ❌ Без поиска
- ❌ Без прогресса обучения
- ❌ Без аккаунтов и авторизации
- ❌ Без экспорта в PDF/другие форматы
- ❌ Без деплоя на сервер (только локальный запуск)

## Визуальные ограничения

- ❌ Никаких кричащих градиентов
- ❌ Никакого неона
- ❌ Никаких тяжёлых теней

## Технические ограничения

- ❌ Не добавлять particles-библиотеки
- ❌ Не использовать postprocessing (bloom, depth of field и т.д.)
- ❌ Не устанавливать Playwright локально (использовать глобальный)
- ❌ Не добавлять новые зависимости без реальной необходимости

## Layout ограничения

```
❌ Запрещено:
<Layout className="bg-ivory"> /* на весь экран */ }

✅ Правильно:
- Фон задаётся body / Three.js сценой
- Контент через подложки: bg-ivory/80 + backdrop-blur
```
